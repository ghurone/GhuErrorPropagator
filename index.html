GhuErrorPropagator

Sobre

Criado por Érick Ghuron, aluno de graduação em Licencitura em Física do Instituo de Física da Universidade de São Paulo (IFUSP), tem o intuito de facilitar a maneira de como propagar incerteza no laboratório.

O código foi escrito totalmente em python e utiliza a biblioteca sympy para realizar o calculo simbólico, o matplotlib para gerar as imagens das equações e o tkinter, biblioteca de interface gráfica padrão do python.

Como usar

A interface do programa é bem simples. Você escreve a função que deseja calcular a incerteza teórica, adiciona os valores das suas variaveis e aperta no botão de propagar.

Claro que se só fosse isso, essa seção nem exisitria. Há várias regrinhas na hora de escrever suas funções e irei lista-las
